# Telegram Finance Bot 

Telegram бот для отслеживания котировков акций, облигаций, валютных пар и финансовых новостей с функцией управления инвестиционным портфелем.

## Основные функции

### Котировки ценных бумаг
- **Российские акции** (Мосбиржа) - SBER, GAZP, YNDX и другие
- **Российские облигации** - ОФЗ, корпоративные, муниципальные
- **Зарубежные акции** (NASDAQ/NYSE) - AAPL, TSLA, GOOGL и другие
- **Подробные графики** и аналитика

### Финансовые новости
- **Экономические новости** - бизнес и финансы
- **Политические новости** - государственные вопросы
- **Новости по тикерам** - актуальные события по компаниям

### Валютные пары
- Основные пары: USD/RUB, EUR/RUB, CNY/RUB
- Любые пользовательские пары: EUR/USD, GBP/USD и другие

### Управление портфелем
- Добавление позиций в портфель
- Отслеживание стоимости инвестиций
- Excel отчеты по портфелю
- История операций

### Отчеты
- Excel отчеты по запрошенным котировкам
- Детализированные отчеты по портфелю
- Автоматическое формирование графиков

## Установка и запуск

### Предварительные требования
- Python 3.8+
- Telegram Bot Token (от @BotFather)
- Finnhub API Key (для зарубежных акций)
